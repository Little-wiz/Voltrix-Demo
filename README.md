# Ehiz Mogaji Electrical — Website

Business website for Ehiz Mogaji Electrical, a green energy & electrical services company based in Nigeria.

## Tech Stack

- React 18
- Plain CSS (per-component stylesheets)
- EmailJS (contact form)

## Getting Started

```bash
npm install
npm start
```

Runs at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Sections

Navbar · Hero · Services · About · Portfolio · Testimonials · Contact · Footer

## Deployment

```bash
npm i -g vercel && vercel
```
